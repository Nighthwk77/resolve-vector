# Contributing

Resolve Vector's application source is developed privately. This public repository does not accept source-code pull requests.

Useful contributions include:

- reproducible bug reports;
- focused feature requests;
- documentation corrections;
- compatibility reports for macOS versions and AI providers.

For security issues, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

By participating, do not submit content you do not have permission to share. Remove credentials, private code, personal data, and proprietary logs before posting.
